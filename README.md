# Bienvenue sur "Mes photos par moi"

Ce projet est un site web personnel où je partage mes photographies capturées à travers le monde. Chaque image est une tentative de capturer des moments uniques et spéciaux.

## Contenu

Le site est divisé en plusieurs sections principales :

- **Accueil** : Présente une introduction au site et à mes photographies.
- **Galeries** : Regroupe mes photos par catégories comme l'art, l'urbain, la nature, les événements, et autres.
- **À propos** : Donne une brève présentation de moi-même en tant que photographe.

## Navigation

La navigation principale du site comprend les éléments suivants :

- **Accueil** : Vous ramène à la page d'accueil.
- **Galeries** : Vous permet de filtrer les photos par catégorie.
- **À propos** : Vous donne des informations sur le photographe (moi).

## Fonctionnalités

- **Filtres de Galerie** : Vous pouvez sélectionner des catégories spécifiques pour voir seulement les photos correspondantes.
- **Modalités** : Cliquez sur une photo pour l'agrandir et la voir en détail.
- **Téléchargement** : Téléchargez une copie haute résolution de chaque photo en cliquant sur le lien de téléchargement dans la modalité.
